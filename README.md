## Setup
To run this project, install it using termux:

```
$ pkg install git
$ pkg install python2
$ pip2 install requests
$ git clone https://github.com/kyzenx/SMBF-tools
$ cd
$ chmod +x install.sh
$ chmod +x smbf.py
$ bash install.sh
$ python2 smbf.py [username] [no.threads] [password]
```
